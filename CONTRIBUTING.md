# Contributing to Cranpose

We welcome contributions!

## Quick Start
1.  **Run Tests**: `cargo test`
2.  **Run Demo**: `cargo run -p desktop-demo`

## Important: Agent Guidelines
Please read [`AGENTS.md`](AGENTS.md) before submitting code. It contains our strict engineering standards:
- No `unsafe` code.
- Zero `clippy` warnings allowed.
- Architecture over shortcuts.

## Submission Checklist
- [ ] `cargo fmt`
- [ ] `cargo check`
- [ ] `cargo clippy` (must be clean)
- [ ] Add `///` docs to new public APIs.

Thanks for helping us build Cranpose!
